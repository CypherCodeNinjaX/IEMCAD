# ⚡ IEMCAD v3.0 — Circuit Design & Simulation Platform

> **Professional circuit design, simulation & diagnostics web application**  
> Built for **IEM Salt Lake, Kolkata** — Institute of Engineering & Management

## 🚀 Features

- **120+ Components** — Passive, Active, Diodes, LEDs, Power, 15 Meter types, Digital Logic, Sensors, ICs
- **Smart Wire Routing** — Magnetic pin snapping with visual indicators
- **Real-time Simulation** — Live V/I/P readings, LED glow, motor RPM, oscilloscope
- **15 Meter Types** — Voltmeter, Ammeter, Ohmmeter, Wattmeter, Multimeter, Oscilloscope, Freq Counter, Cap Meter, Inductance Meter, PF Meter, Energy Meter, Lux Meter, Temp Meter, dB Meter, Logic Probe
- **Editable Values & Units** — Change any component value and unit in real-time
- **Smart Diagnostics** — Shows WHY current isn't flowing with fix suggestions
- **Sound Effects** — Web Audio API synthesized sounds for every interaction
- **Easter Eggs & Mini Games** — Type hidden words, play circuit quizzes
- **Plugin Architecture** — Extensible for future updates

## 🛠 Tech Stack

React 18 • Vite 5 • Web Audio API • SVG Rendering • CSS Animations

## 📦 Getting Started

```bash
npm install
npm run dev
```

## ⌨️ Shortcuts

V=Select • W=Wire • D=Delete • R=Rotate • Del=Remove • Ctrl+Z/Y=Undo/Redo • Alt+Drag=Pan • Scroll=Zoom
